## Usage

Simply open `index.html` in a modern web browser. All pages are static and work without a server.

For best results, ensure you have an active internet connection as the site loads:
- Google Fonts (Playfair Display, Inter)
- Tailwind CSS (via CDN)
- AOS library (via CDN)
